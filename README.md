# SynaptiQ Skill

This repository hosts a Google AI Edge Gallery skill for SynaptiQ interview prep.

## Repository layout

```txt
my-custom-coach/
  SKILL.md
```

## Raw import URL

After pushing this repository to `main`, use:

```txt
https://raw.githubusercontent.com/selokarabhishek/synaptiq_skill/main/my-custom-coach/SKILL.md
```

## iPhone note

If `Load skill from URL` fails in Google AI Edge Gallery on iPhone, open the raw URL in Safari, save the file to Files, then use `Load skill from Local`.
